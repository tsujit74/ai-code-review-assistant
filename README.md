# AI-Powered Code Review Assistant

## Overview
Build a production-oriented AI Code Review Assistant that allows users to upload source code and receive structured AI-generated code reviews.

## Tech Stack
- **Frontend:** Next.js, TypeScript, TailwindCSS, shadcn UI
- **Backend:** NestJS, TypeScript
- **Database:** PostgreSQL, Prisma ORM
- **Authentication:** JWT
- **AI Providers:** OpenAI, LM Studio, OpenRouter, Ollama (bonus)

## Features
- Authentication (Register, Login, Logout)
- Project Management (Create, List, Delete)
- Code Upload (ZIP), Code Explorer, File Preview
- AI Review Engine (Security, Performance, Quality)
- Review History & Search
- AI Chat With Code (bonus)
